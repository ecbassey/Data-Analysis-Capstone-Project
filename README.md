# Data-Analysis-Capstone-Project

## OPTION 1: Airline Performance & Delay Intelligence System



### Problem Statement
This project analyzes how COVID-19 impacted airline performance in the U.S., what recovery look like through delays, cancellations, and airport congestion patterns. The project would analyze flight performance, analyze airline, analyze airport congestion and identify delay patterns.


### Tools/Skills use:
- Data ingestion: scraping, CSV
- Data Transformation: BigQuery
- Data Storage: SQL
- Data cleaning & analysis: Python / Pandas
- Software design: OOP
- Visualization: matplotlib / seaborn / GeoPandas


### Insights and Solutions:
This project aims to transform raw flight data into meaningful insights that help airlines, airports, and stakeholders better understand and manage delays and cancellations. This project aims to uncover:
- Which airline is most reliable
- What month has the highest delays - Time-based trends in delays (months, seasons, etc.)
- Did the industry recover after COVID
- Which airports are bottlenecks
- What factors most influence delays
- Delay and cancellation patterns across different airlines


### Business and community impact of this anlysis:
#### Airline Companies
Gain insights into delay patterns, operational inefficiencies, and performance trends, helping them improve scheduling, reduce delays, and optimize resources.

#### Airport Authorities
Identify congestion points, peak traffic times, and bottlenecks to improve airport operations and capacity planning.

#### Government & Aviation Regulators
Monitor industry performance, assess the impact of disruptions (like COVID-19), and make informed policy or infrastructure decisions.

## OPTION 2: Fertility Rate Intelligence System

### Problem Statement
This project aims to build a system that integrates and analyzes fertility-related data to uncover trends, identify driving factors, and support data-driven insights into global population patterns.

### Insights and Solutions:
This project aims to uncover:
- How has fertility rate changed globally over time
- Which countries have the highest/lowest rates
- Is fertility declining as GDP rises
- Does female education reduce fertility rates
- Are developed countries below replacement level
